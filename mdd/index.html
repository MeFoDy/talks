Клавиатурные сокращения

  • Windows: Ctrl + C
  • macOS: ⌘ + C
document.addEventListener("keydown", function(e) {
    var isMac = window.navigator.platform.match("Mac");
    if ((isMac ? e.metaKey : e.ctrlKey) && e.keyCode == 67) {
        e.preventDefault();
        // обработка Ctrl/Cmd + C
    }
}, false);

Что делать?

Тестировщики

  • умеют лучше;
  • не пропустят баг, потому что "фича";
  • медленное появление фичи у пользователей.

Отдельное устройство для тестирования

  • разработчик не пропустит баг сам;
  • дорого;
  • поддержка инфраструктуры.

Две параллельные системы

  • разработчик не пропустит баг сам;
  • относительно дёшево;
  • поддержка инфраструктуры.

Разрабатывать сразу на Windows

  • дёшево;
  • можно на сдачу купить разработчику второй монитор;
  • баги видно сразу;
  • разглаживаются морщины.
