Строгий режим

"use strict";
function strictFunction() {
    "use strict";
}
  • Преобразовывает ошибки в исключения
  • Запрещает использование with
  • Не добавляет новых переменных в контекст из eval
  • "Чинит" ошибки предыдущих версий стандарта
  • Запрещает использование ключевых слов из будущих версий стандарта
